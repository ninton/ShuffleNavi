# ShuffleNavi

GooglePlay  
https://play.google.com/store/apps/details?id=jp.co.ninton.shufflenavi

GitHub  
https://github.com/ninton/ShuffleNavi

The navigator application for a shuffle to do a realistic card.

When a good card comes when playing cards, isn't a shuffle done well? I think a shuffle is often done when a bad card comes.

So a shuffle do a card using a shuffle navigator, please.

The one which gathered a card is called a deck.
It's called a cut to share a deck with 2 by a suitable place and replace the top and the bottom.

Step 1
A deck is cut once.

Step 2
Please distribute a deck to 10 packets with screen indication of a shuffle navigator. It doesn't become equal, but the card count in each packet is normal with this.

Step 3
Please collect 10 packets. It's cut once again at the end.

Well, a game start.

# シャッフルナビ

GooglePlay  
https://play.google.com/store/apps/details?id=jp.co.ninton.shufflenavi

GitHub  
https://github.com/ninton/ShuffleNavi

リアルなカードをシャッフルするためのナビアプリです。

トランプをするとき、いいカードがくると、よくシャッフルされていないな？ばらばらなカードがくると、よくシャッフルされているな、と思います。

そこで、シャッフルナビを使って、カードをシャッフルしてください。

カードをまとめたものをデッキと呼びます。
カットとは、デッキを適当な場所で2つに分けて、上下を入れ替えることをいいます。

ステップ1
デッキを1回カットします。

ステップ2
シャッフルナビの画面表示にしたがって、デッキを10個の山に配ってください。各山のカード枚数は均等にはなりませんが、これで正常です。

ステップ3
10個の山を集めてください。最後にもう1回カットします。

さあ、ゲームスタートです。
